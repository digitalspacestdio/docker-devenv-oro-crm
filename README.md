# OroCommerce Development Environment

Put your code to the `www` folder
```bash
git clone git@github.com:oroinc/orocommerce-application.git www
```

Build the stack
```bash
docker-compose build
```

Start the stack
```bash
docker-compose up
```

Stop the stack
```bash
docker-compose down
```

Destroy the whole data
```bash
docker-compose down -v
```
